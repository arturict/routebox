# RouteBox

RouteBox is a minimal route manager for Coolify Traefik dynamic YAML.

- `app/` contains the Flask product UI and API.
- `landing/` contains the public marketing site deployed at `https://routebox.cloud.arturf.ch`.

The app is designed for homelabs and small infrastructure teams that use Coolify
as the public reverse proxy but still expose services running outside Coolify.
